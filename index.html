插入排序

Mar 30th, 2014 

#include <iostream>
using namespace std;
//参数为数组和其大小
void insert(int *data,int n)  
{
    int target;
    for(int i=1; i<n; i++) 
    {
        int j = i;
        target = data[i]; //将要排序的元素
        while(j > 0 && target < data[j-1])  //排序的条件，j大于0，并且将要排序的元素要找到从左边起第一个大于它的元素，放在这个元素的左边，
        {
            data[j] = data[j-1];
            j--;
            //后边的元素整体后移一个
        }
        data[j] = target;
    }
    for(int i = 0;i < n; i++)
        cout << data[i] <<",";
}

void main()
{
    int data[]={6,4,2,3,1,7,9};
    int n=7;
    cout << "排序前：";
    for(int i = 0;i < 7; i++)
        cout << data[i] <<  ",";

    cout << "排序后：";
    insert(data,n);
    cout << endl;
}
//平均时间复杂度O(n2)

int arr[15]={0,1,2,2,2,2,3,3,3,4,5,6,7,8,9};

if(a==-1)
    cout<<"出错 "<<endl;
else 
    cout<<"数组下标："<<a<<endl;
return 0;
}

int search(int arr[],int k,int n)
{
    int start,end,mid;
    start=0;
    end=n-1;
    while(start<=end)   //判断条件
    {
        mid=(start+end)/2;  //取中间值
        if(k==arr[mid])
        {
            while(k==arr[mid])
            {
                mid++;
            }
    /*
        加上while可以获取最右边的第一个出现的索引，不加while为最左边第一次出现的索引
    */
            return mid-1;
        }
        else if(k<arr[mid])  //继续往左查找
            end=mid-1;
        else
            start=mid+1;  //往右查找

    }
return -1;
}

Nov 24th, 2013

最近一周直到昨天都一直在折腾uip，昨天，就是昨天终于结束了，其中的酸甜苦辣啊。
至于uip是什么，我就不废话了。我主要来记录一下我的移植过程。最主要的目的是先实现一个最简单的功能，即能够ping通，再实现tcp的连接。

需要的东西

单片机开发板一块如上图单片机选型，rom和ram都最好大一点。我的是stc90c516RD+，至于网络模块，当然是选择比较便宜的enc28j60模块，淘宝上有现成的做好的模块，外接5v电源就可以，因为其接入了一个降压模块ams1117。此网络模块对电压要求比较高，我的开发板输出的电压差不多是4.8，4.9v的样子，但连上网络模块之后始终网口不发亮，一测芯片的电压，就差一点达不到3.3v，没办法，只好找了个5v的电源来给其供电。另外我的单片机rom61k,ram1280字节。空间还是挺大的。开发环境用的是keil。至于uip可以移植到多大空间的地方，我也不是很清楚，反正我这个最后还是成功了。

网络连不上

Nov 12th, 2013

晚上回到宿舍，一直连不上网，之前就遇到过这问题，但是糊里糊涂的解决的，今天说什么也得弄好了。
问题：开机即可获得以个169.254.xx开头的ip，使用了固定ip后也是这样，这样很明显是不会连上网的，而且子网掩码还是255.255.0.0，没办法，在网上找了找，大致有下面几种方法，最有效的是第三种。
169.254.X.X是Windows操作系统在DHCP信息租用失败时自动给客户机分配的IP地址。意思就是在你开机之后，网卡自动获取不到合适的ip，所以会自动得到一个微软自带的ip，但是这个会影响下面的正常上网。
解决办法：
1.分配固定的ip地址，首先你要看一下别人可以正常上网的ip地址，网关，dns等。
2.重启下机器或者网卡。
3.使用下面的命令

 ipconfig  /renew 清除ip缓存
 ipconfig /flushdns 清除dns缓存
 arp -d* 清除arp缓存
